# MediStock

MediStock es un sistema web para controlar el inventario de medicamentos en una farmacia hospitalaria.

## Funcionalidades principales

- Registro de medicamentos
- Control de inventario
- Registro de salidas
- Alertas de caducidad

## Tecnologías

- Frontend
- Backend
- Base de datos

Proyecto desarrollado para la materia de Ingeniería de Software.
